Welcome!

Problem: Whenever your crush changes their photo, you lose access to the old profile photo (considering you two are not friends and privacy...).
Solution: Keep saving all the photos as they appear.
This is a facebook profile stalking app. What it does is, visit the profile of everyone on the stalk list every hour and check if the profile picture has been updated. If yes, it saves the URL to the profile picture for you to easily view later by visiting your list here. All previous profile pictures are also stored for you to see in an album form.

Start Stalking

You need to create a stalklist first. OR Enter the name of list you created before.

REMEMBER!

The name of the list is your only way in or anyone else's way in. Lists are not password protected. So name them well.

FAQ!

Only the profile picture?
Yes, currently only indexing of the profile picture is supported. I will add more if this becomes worth it.

Where are images stored?
I just index the URLs of the images on my server, the images are never saved to my server nor do I have space for all those selfies.

How does it work?
I run a headless browser at my server. A cron is running which visits each of the profiles in everyones' stalk list and parses the profile picture and saves them. This was a weekend project for testing how quick it is to make something using Firebase Service. I am impressed so far.

Who pays for the server?
I do currently.
